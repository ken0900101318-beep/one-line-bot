## 📋 完成後要做的事

### 1. 設定 LINE Bot Webhook

**等 Railway 部署完成後：**

1. **取得你的 Railway 網址**
   - 格式：`https://one-line-bot-production.up.railway.app`
   - 可在 Railway Dashboard 查看

2. **前往 LINE Developers Console**
   - https://developers.line.biz/console/

3. **Messaging API → Webhook settings**

4. **填入 Webhook URL**
   ```
   https://你的Railway網址.up.railway.app/webhook
   ```

5. **啟用 Use webhook**

6. **驗證 Webhook**（點擊 Verify）

7. **關閉 Auto-reply messages**

## 🆘 如果遇到問題

### Railway 部署失敗
```bash
cd ~/.openclaw/workspace/one-line-bot
railway logs
```

### LINE Bot 不回應
- 檢查 Webhook 是否驗證成功
- 檢查 Railway logs 有無錯誤
- 確認環境變數是否正確

### AI 回覆失敗
- 檢查 ANTHROPIC_API_KEY 是否正確
- 檢查 API 額度是否足夠

### Google Sheets 讀取失敗
- 確認 Sheets 已設定公開分享
- 確認 CSV 網址正確
- 檢查網路連線
